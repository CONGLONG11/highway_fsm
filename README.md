# 基于模糊增强有限状态机的高速公路自动驾驶决策规划系统

> 毕业设计项目 | 基于 CARLA 仿真平台

## 一、项目简介

本项目面向**高速公路场景**，设计并实现了一套完整的自动驾驶
**"感知—决策—规划—控制"**四层架构系统。系统在 CARLA 0.9.15
仿真环境中运行，能够实现自适应巡航、智能变道、紧急避让等功能。

## 二、创新点说明

### 创新点 1：模糊逻辑增强的有限状态机决策

传统 FSM 使用硬阈值（如 `if distance < 30`），在阈值边界存在
**决策抖动**问题。本文引入模糊逻辑（Fuzzy Logic），将连续的
物理量（距离、速度差）映射为模糊语义（"近"、"中"、"远"），
通过模糊推理得到连续的**变道意愿度**，再结合代价函数做出
最终决策。有效解决了阈值边界的决策震荡问题。

### 创新点 2：基于人工势场的动态轨迹规划

传统五次多项式仅考虑起终点约束，生成的轨迹不感知障碍物。
本文将**人工势场 (APF)** 与五次多项式结合：
- 目标车道产生**引力**
- 周围车辆产生**斥力**
- 势场梯度用于修正五次多项式生成的基准轨迹
使变道轨迹能主动避开危险区域，提高了规划的安全性。

### 创新点 3：自适应增益调度的 Stanley 控制器

标准 Stanley 控制器使用固定增益参数，无法同时兼顾
高速稳定性和低速灵活性。本文提出**增益调度策略**：
- 根据车速动态调整 `k_stanley` 增益
- 根据轨迹曲率动态调整预瞄距离
- 引入一阶低通滤波消除转向抖动
实现了全速域的平稳跟踪。

## 五、参数配置
所有可调参数集中在 config/config.yaml 中：
# 目标车速 (km/h)
target_speed: 100.0

# 模糊逻辑参数
fuzzy:
  dist_close: 20.0
  dist_medium: 40.0
  dist_far: 60.0

# 势场参数
apf:
  k_attract: 1.0
  k_repulse: 50.0
  repulse_range: 30.0

# 控制器参数
controller:
  k_stanley_base: 0.5
  lookahead_min: 3.0
  lookahead_max: 15.0
  steer_filter_alpha: 0.3

## 七、实验指标
系统评估使用以下量化指标：

平均横向误差 (m)	        轨迹跟踪精度
最大横向加速度 (m/s² )	    乘坐舒适性
变道成功率 (%)	            决策有效性
平均车速 (km/h)	            通行效率
碰撞次数	                安全性
决策抖动次数	            FSM 稳定性
